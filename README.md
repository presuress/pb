# 房屋租赁系统
<h5>📝 项目简介</h5>

本项目旨在构建一个高效、安全且易于使用的房屋租赁平台，旨在解决传统租房流程中信息不对称、沟通繁琐的问题。系统采用 前后端分离架构，利用 Spring Boot 提供稳健的业务逻辑处理能力，结合 Vue.js 实现响应式用户界面，为房东与租客提供无缝的交互体验。

开发工具 IntelliJ IDEA、MySQL、Navicat、Node.js、JDK、Maven、VSCode

<h5>📦 安装部署</h5>

<h2>一、数据库部署：</h2>

1.打开Navicat，连接到MySQL服务器

2.创建新数据库(CREATE DATABASE housexxx;)

3.导入数据找到项目提供的.sql文件,在Navicat中打开该SQL文件并执行所有SQL语句

<h2>二、后端部署：</h2>

<p>1.Maven配置</p>

IDEA中配置：文件 → 设置 → 构建、执行、部署 → 构建工具 → Maven

配置路径：

Maven主路径：选择解压后的Maven文件夹

用户设置文件：选择conf/settings.xml

本地仓库：创建新目录作为本地仓库

2.JDK配置

打开IDEA：文件 → 项目结构 → 项目设置 → 项目

设置SDK为JDK 17或更高版本

若无JDK，点击"添加SDK" → "下载JDK"

3.项目依赖下载

导入SpringBoot项目

打开Maven侧边栏（IDEA右侧），点击刷新按钮下载依赖

<h5>📂📂📂导入SpringBoot项目后要修改你的MySQL用户名和你的MySQL用户名
  之后启动后端服务</h5>

<h2>三、前端部署</h2>
1.IDEA 导入前端项目

文件 → 项目结构 → 模块

点击加号 → 导入模块 → 选择前端项目文件夹

选择"从现有项目创建模块" → 完成创建

2. 配置NPM运行环境
点击IDEA右上角下拉菜单 → 编辑配置

点击加号 → 选择npm

脚本输入框输入：dev  (点击确定)

3.启动

cd ../vue  # 替换为你的前端项目文件夹名
npm install

依赖安装完成后，点击IDEA右上角绿色三角形运行项目

访问控制台显示的链接
